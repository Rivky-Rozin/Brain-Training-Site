# פרויקט גמר

פרויקט זה הוא חלק מפרויקט הגמר של הקורס.

## התקנה

1. התקן את התלויות:
```bash
npm install
```

2. הפעל את השרת:
```bash
npm start
```

## מבנה הפרויקט

- `client/` - קוד הצד-לקוח
- `server/` - קוד הצד-שרת
  - `services/` - שירותים
  - `routes/` - נתיבי API
  - `controllers/` - בקרים
  - `db/` - הגדרות בסיס הנתונים
